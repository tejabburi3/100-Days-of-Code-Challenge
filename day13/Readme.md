The **Higher or Lower Game** is a console-based Python game where:
- Two random options are displayed
- The player guesses which option has a **higher value**
- The score increases for every correct guess
- The game ends when the player makes a wrong guess

The game uses **multiple datasets** such as:
- Google search popularity
- Social media followers
- YouTube subscribers
- IMDb movie ratings
