This project is a **command-line calculator** built using Python.

## â–¶ How the Calculator Works
1. User enters the first number.
2. User selects an operator.
3. User enters the second number.

5. User can:
   - Continue using the previous result, or
   - Start a new calculation, or
   - Exit the program.
